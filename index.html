Traning

NovusInfo provides online interactive training and hands on lab experience on Cloud Computing specifically targeted to AWS public cloud which will enable candidate to pass the AWS certification examinations. Our trainers are experts and have over decade experience in IT industry. Following are the AWS certifications targeted for Novusinfo training

  • 1. AWS Certified Solutions Architect
    • a. Associate
    • b. Professional
  • 2. AWS Certified Developer - Associat
  • 3. AWS Certified SysOps Administrator - Associate
  • 4. AWS Certified DevOps Engineer - Professional

Apart from AWS cloud Novusinfo provides training and hands on lab targeted to generic DevOps. Following are covered in DevOps Course

1. DevOps process and steps - Industry Best practices on Code Configuration Management, Continuous integration, Continuous Delivery, Infrastructure configuration management , automation and Continuous infrastructure enhancements, Monitoring,Auditing, Reporting, Alerting and Log Aggregation

2. Tools and Techniques covered.

  • a. Code Configuration Management - Tools-Git, SVN Techniques- Branching,Merging strategies and other configuration management techniques
  • b. Continuous integration - Tools- Jenkins and Atlassian Bamboo Techniques - Continuous Integration build, On demand build, Deployment artifacts repos such as S3 & Nexus and deployment builds, Automated test execution
  • c. Infrastructure configuration management, automation and Continuous infrastructure enhancements - Tools- Chef & Puppet, Container-Docker,Terraform, Cloud Formation, Shell scripting, SDK for different cloud providers
  • d. Monitoring - Nagios,Zabbix CloudWatch and Cacti
  • e. Log Aggregation - Logstash, Splunk and Loggley and services specific to each Cloud providers
  • f. Performance measurement - PSI score, Google Developers tools and Newrelic
  • g. Others - Apache,Tomcat, Nginx,Jboss and other open source based app and web servers

After the training Novusinfo provides the pilot project to work on so that real time exposure can be gained. Once pilot project is completed candidate will be provided with placement assistance

Apart from individual courses, NuvusInfo provides in-house corporate training as well.

Following are the other areas NovusInfo will provide interactive and hands on training

  • Mobile Development
  • Big data and analytics
  • Full stack development using NodeJS, AngularJS and Mongo db
  • Advanced Java, .Net
  • Advanced database concepts using Oracle and MS SQL Server
  • Software Testing
  • SAP
