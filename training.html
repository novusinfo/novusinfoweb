Training Services

AWS Certification Training:

NovusInfo provides online interactive training and hands on lab experience on Cloud Computing specifically targeted to AWS public cloud which will enable candidate to pass the AWS certification examinations. Our trainers are experts and have over decade experience in IT industry.

Following are the AWS certifications targeted for Novusinfo training

- AWS Certified Solution Architect

- AWS Certified Developer - Associate

DevOps Training:

Apart from AWS cloud Novusinfo provides training and hands on lab targeted to generic DevOps. Novusinfo provides training on 'DevOps process and steps' including Industry's best practices on Code Configuration Management, Continuous integration, Continuous Delivery, Infrastructure configuration management , automation and Continuous infrastructure enhancements, Monitoring,Auditing, Reporting, Alerting and Log Aggregation

- Tools and Techniques covered .

a. Code Configuration Management - Tools-Git, SVN Techniques- Branching,Merging strategies and other configuration management techniques

b. Continuous integration - Tools- Jenkins and Atlassian Bamboo Techniques - Continuous Integration build, On demand build, Deployment artifacts repos such as S3 & Nexus and deployment builds, Automated test execution

c.Infrastructure configuration management, automation and Continuous infrastructure enhancements - Tools- Chef & Puppet, Container-Docker,Terraform, Cloud Formation, Shell scripting, SDK for different cloud providers

d. Monitoring - Nagios,Zabbix CloudWatch and Cacti

e. Log Aggregation - Logstash, Splunk and Loggley and services specific to each Cloud providers

f. Performance measurement - PSI score, Google Developers tools and Newrelic

g. Others - Apache,Tomcat, Nginx,Jboss and other open source based app and web servers
