DevOps Training

Apart from AWS cloud Novus InfoSolutions provides training and hands on lab targeted to generic DevOps. Novus InfoSolutions provides training on 'DevOps process and steps' including Industry best practices on Code Configuration Management, Continuous integration, continuous Delivery, Infrastructure configuration management , automation and Continuous infrastructure enhancements, Monitoring,Auditing, Reporting, Alerting and Log Aggregation.

Tools and Techniques covered

  • Code Configuration Management - Tools-Git, SVN Techniques- Branching,Merging strategies and other configuration management techniques.
  • Continuous integration - Tools- Jenkins and Atlassian Bamboo Techniques - Continuous Integration build, On demand build, Deployment artifacts repos such as S3 & Nexus and deployment builds, Automated test execution.
  • Infrastructure configuration management, automation and Continuous infrastructure enhancements - Tools- Chef & Puppet, Container-Docker,Terraform, Cloud Formation, Shell scripting, SDK for different cloud providers.
  • Monitoring - Nagios,Zabbix CloudWatch and Cacti.
  • Log Aggregation - Logstash, Splunk and Loggley and services specific to each Cloud providers.
  • Performance measurement - PSI score, Google Developers tools and Newrelic.
  • Others - Apache,Tomcat, Nginx,Jboss and other open source based app and web servers.
